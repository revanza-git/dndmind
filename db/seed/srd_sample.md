# Ability Checks

An ability check tests a character's or monster's innate talent and training in an uncertain situation. The DM calls for an ability check when a character attempts an action that has a chance of failure.

## Advantage and Disadvantage

Sometimes a special ability, spell, or circumstance says that you have advantage or disadvantage on an ability check, an attack roll, or a saving throw. When that happens, roll a second d20 when you make the roll.

If you have advantage, use the higher of the two d20 rolls. If you have disadvantage, use the lower roll. For example, if disadvantage gives you a 17 and a 5, you use the 5.

If multiple situations affect a roll and each one grants advantage or imposes disadvantage, you do not roll more than one additional d20. If circumstances cause both advantage and disadvantage, they cancel out and you roll one d20.

## Difficulty Classes

The DM sets the difficulty class for an ability check. A typical easy task is DC 10, a moderate task is DC 15, and a hard task is DC 20.

# Combat

## Attack Rolls

When a character makes an attack, the attack roll determines whether the attack hits. Roll a d20 and add the relevant ability modifier and proficiency bonus if applicable. If the total equals or exceeds the target's Armor Class, the attack hits.

## Saving Throws

A saving throw represents an attempt to resist a spell, trap, poison, disease, or similar threat. The DM decides which ability score is used for the saving throw.
